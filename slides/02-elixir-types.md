# Elixir types

Elixir has some basic types:

|type|examples|
|------|------|
| Numeric (integer)| `1` |
| Numeric (float)| `1.0`, `6.022e23` |
| Booleans | `true`, `false`|
| Atoms | `:atom`, `AlsoAnAtom` |
| Strings | `"elixir"` |
| Charlists | `'erlang'` |
| Lists | `[ 1, 2, 3]` |
| Attribute Lists | `[ from: 'me', to: 'you' ]` |
| Tuple | '{1, two, 3}'|
